# youtube-clone-app
Project demo: [youtube-clone-app](https://gabrielluizs1996.github.io/youtube-clone-app/build/)

#Prerequisites
* Node.Js
* Yarn or NPM
* React

# Install

### yarn
```sh
$ git clone https://github.com/gabrielluizs1996/youtube-clone-app.git
$ cd youtube-clone-app
$ yarn
$ yarn start
```

### NPM
```sh
$ git clone https://github.com/gabrielluizs1996/youtube-clone-app.git
$ cd youtube-clone-app
$ npm install
$ npm start
```
